# Bug Report Tracker

## Overview
This repository contains sample bug reports based on issues identified while testing web applications.

## Purpose
The purpose of this project is to demonstrate my ability to:
- Identify defects
- Reproduce bugs consistently
- Write clear and professional bug reports
- Communicate technical issues effectively

## Skills Demonstrated
- Defect Logging
- Analytical Thinking
- Attention to Detail
- QA Documentation
- Communication

## Included
- Multiple bug reports
- Reproducible steps
- Expected and actual results
- Severity and priority classification
