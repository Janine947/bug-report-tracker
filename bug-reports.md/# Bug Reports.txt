# Bug Reports

## BUG_001 - Login button does not respond after entering valid credentials

**Module:** Login  
**Severity:** High  
**Priority:** High  
**Environment:** Chrome on Windows 10

### Steps to Reproduce
1. Open the login page
2. Enter valid email address
3. Enter valid password
4. Click Login

### Expected Result
User should be logged in and redirected to the dashboard.

### Actual Result
Login button does not respond.

### Status
Open

## BUG_004 - Registration form accepts invalid email format

**Module:** User Registration  
**Severity:** Medium  
**Priority:** Medium  
**Environment:** Chrome on Windows 10

### Steps to Reproduce
1. Open the registration page
2. Enter full name
3. Enter invalid email format such as janine@com
4. Enter password
5. Click Register

### Expected Result
System should reject invalid email format.

### Actual Result
Form accepts the invalid email and proceeds.

### Status
Open
